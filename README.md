# AIPPT2 - 像做APP一样做PPT

## 核心理念

把PPT当做APP的页面来设计：
- 每张PPT = 一个页面/屏幕
- 先用ASCII画结构，再填充内容和配图

## 工作流程

```
ASCII框架 → 内容填充 → AI生图 → 成品PPT
```

## 方法论

### ASCII 框架设计

用ASCII画出每张PPT的骨架，明确：
- 信息层级（标题、副标题、正文）
- 空间布局（左右分栏、上下分区）
- 元素占比（文字区 vs 图片区）

### 常用模板

| 类型 | 用途 |
|------|------|
| cover | 封面页 |
| content | 正文页（图文混排） |
| section | 章节分隔页 |
| quote | 金句页 |
| end | 结束页 |

### 模板示例

**封面页 cover**
```
┌────────────────────────────────────────┐
│                                        │
│                                        │
│         【主标题】                      │
│         副标题说明                      │
│                                        │
│                                        │
│                           作者 / 日期  │
└────────────────────────────────────────┘
```

**正文页 content**
```
┌────────────────────────────────────────┐
│  标题                                  │
├───────────────────┬────────────────────┤
│                   │                    │
│   [配图区域]       │   · 要点一         │
│                   │   · 要点二         │
│                   │   · 要点三         │
│                   │                    │
└───────────────────┴────────────────────┘
```

**金句页 quote**
```
┌────────────────────────────────────────┐
│                                        │
│                                        │
│      "这里是金句内容"                   │
│                                        │
│                          —— 来源       │
│                                        │
└────────────────────────────────────────┘
```
